## 🎯 Descripción General

**Talento Tech - React JS** es un curso cuatrimestral intensivo diseñado para formar desarrolladores frontend capaces de crear aplicaciones web interactivas, dinámicas y escalables utilizando **React**, la biblioteca más demandada en la industria tecnológica actual.

React se ha consolidado como un estándar de facto en el desarrollo frontend, permitiendo construir interfaces de usuario modernas que integran:

- ✨ Componentes reutilizables y escalables
- 🎮 Interactividad dinámica y fluidez
- 🔄 Gestión eficiente del estado aplicacional
- 🛣️ Navegación segura con rutas protegidas
- 🔐 Sistemas de autenticación robustos
- 📦 Operaciones CRUD completas
- 🎨 Diseño responsive y atractivo

Este curso no solo enseña sintaxis y APIs, sino que prepara a los estudiantes para enfrentar desafíos reales en equipos profesionales de desarrollo web.

---

### Competencias Clave

A nivel educativo, la enseñanza de React promueve competencias clave:

- Modularización de aplicaciones
- Implementación de operaciones CRUD
- Manejo de lógica de autenticación
- Despliegue en entornos de producción

Estas habilidades incrementan significativamente la empleabilidad en un mercado laboral donde startups, pymes y grandes empresas requieren soluciones digitales ágiles.

## 🎓 Objetivos

### Objetivo General

Brindar a los estudiantes los conocimientos y habilidades necesarios para diseñar y desarrollar aplicaciones web interactivas con ReactJS, abordando desde la creación de componentes hasta la gestión de estado, el enrutamiento y la autenticación de usuarios. 

El curso tiene como propósito que los participantes integren todas estas competencias en un **proyecto final funcional (un e-commerce)**, aplicando operaciones CRUD, manejo de formularios y rutas protegidas. Además, busca fomentar el aprendizaje de buenas prácticas de desarrollo frontend, el trabajo con librerías de estilización modernas y la capacidad de desplegar proyectos en entornos productivos.

## 🚀 Instalación y Setup

### Paso 1: Clonar el Repositorio

```bash
git clone https://github.com/tu-usuario/talent-tech-tp.git
cd talent-tech-tp
```

### Paso 2: Instalar Dependencias

```bash
npm install
```

### Paso 3: Iniciar el Servidor de Desarrollo

```bash
npm run dev
```

La aplicación estará disponible en `http://localhost:5173`

### Paso 4: Abrir en tu Navegador

- Abre tu navegador web
- Navega a `http://localhost:5173`
- ¡Listo! Ya estás usando la aplicación

## 🛍️ Proyecto Final Integrador

### Descripción

El objetivo del proyecto final es que los estudiantes **apliquen todos los conceptos aprendidos** durante el curso de ReactJS en la creación de una **aplicación de e-commerce funcional**. 

El proyecto debe integrar funcionalidades clave que simulan un ambiente profesional real.

### Funcionalidades Requeridas

✨ **Gestión de Productos**
- Listar productos disponibles
- Ver detalle de cada producto
- Agregar productos al carrito

🔐 **Autenticación de Usuarios**
- Sistema de login/logout
- Rutas protegidas
- Sesión de usuario

📦 **Carrito de Compras**
- Agregar/eliminar productos
- Actualizar cantidades
- Cálculo de totales

✏️ **CRUD de Productos** (Panel administrativo)
- Crear nuevos productos
- Leer/listar productos
- Actualizar información
- Eliminar productos

💾 **Persistencia de Datos**
- Integración con API o Firebase
- Almacenamiento local (localStorage)

🎨 **Interfaz de Usuario**
- Diseño responsivo
- Experiencia fluida
- Elementos visuales atractivos

### Buenas Prácticas

- ✓ Usa componentes funcionales
- ✓ Sigue el patrón contenedor-presentacional
- ✓ Mantén el código limpio y bien documentado
- ✓ Valida formularios antes de enviar
- ✓ Maneja errores apropiadamente
- ✓ Comitea cambios regularmente con mensajes claros
- ✓ Prueba tu código en diferentes navegadores
